# 🎉 EventHub - Event Management System

A modern full-stack Event Management System built using the MERN Stack. This application allows users to discover events, book tickets, manage their profiles, and enables administrators to manage events, categories, bookings, and users through a secure admin dashboard.

## 📌 Project Overview

EventHub is designed to simplify event management by providing an easy-to-use platform for both users and administrators.

Users can:

- Create an account
- Browse upcoming events
- Search and filter events
- View detailed event information
- Book events
- Manage bookings
- Update profile

Administrators can:

- Manage users
- Create and manage events
- Manage categories
- Approve or reject bookings
- View dashboard statistics

# ⚙️ Installation

## Clone Repository

```bash
git clone <repository-url>
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Run backend:

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
