# 🎯 QuizHub - MERN Stack Quiz Platform

QuizHub is a full-stack MERN application that allows users to create, manage, and participate in quizzes. The platform provides secure authentication, quiz management, team creation, leaderboard functionality, and a clean responsive user interface.

## Backend Setup


