## 📌 Project Overview

Color Guessing Game is a simple frontend game where players must identify the correct color based on the RGB value displayed on the screen.

The game generates random colors and challenges the user to select the matching color box.

---

## ✨ Features

* Random RGB color generation
* Interactive color selection
* Score tracking system
* Best score storage using Local Storage
* Easy Mode (3 colors)
* Hard Mode (6 colors)
* Auto next round after correct answer
* Reset score functionality
* Responsive design
* Modern glassmorphism UI
* Hover animations and smooth transitions
